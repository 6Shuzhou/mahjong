# 麻将概率统计

基于蒙特卡洛模拟的麻将概率统计项目。使用 136 张标准牌（34 种 × 4 张，不含花牌）。

---

## 脚本说明

| 文件 | 功能 |
|:---|:---|
| `1.py` | **打烂向听数分布**：模拟随机 14 张牌距离"打烂"和牌型的向听数分布 |
| `2.py` | **打烂+7字牌达成率**：模拟从 13 张起手开始，通过摸牌弃牌达成"打烂且包含 7 种不同字牌"的成功率 |
| `3.py` | **七对子达成率**：模拟从 13 张起手开始，通过摸牌弃牌达成七对子和牌的成功率 |
| `hand_quality.py` | **起手好牌概率**：统计多种起手"好牌"条件的出现概率 |
| `tenpai.py` | **天听概率**：计算起手 13 张即听牌的概率（支持特殊字牌顺子规则） |
| `Distribution.ipynb` | 相关的分布分析 Notebook |

---

## 核心统计结果

### 一、起手好牌概率（`hand_quality.py`，100 万次模拟）

以下 8 个条件满足其一即视为"好牌"：

| 条件 | 概率 |
|:---|---:|
| 起手恰好 4 对 | **2.06%** |
| 起手恰好 5 对 | **0.11%** |
| 起手有一个暗杠（某牌 4 张） | **0.18%** |
| 起手有两个暗刻 | **0.17%** |
| 起手一个暗刻 + 两个对子 | **1.86%** |
| 起手某种花色 ≥ 8 张（清一色/混一色潜力） | **1.68%** |
| 起手有 5 个不同字牌 | **3.09%** |
| 起手有超过 5 个字牌（≥ 6 张） | **2.73%** |
| **至少满足一个条件** | **9.56%** |
| **不满足任何条件** | **90.44%** |

### 二、起手天听概率（`tenpai.py`，50 万次模拟）

规则说明：东南西北任意 3 张不同的字牌可组成顺子，中发白 3 张可组成顺子。

| 和牌型 | 概率 |
|:---|---:|
| **天听总概率** | **0.0196%**（约 1/5100） |
| 一般型天听 | 0.0180% |
| 七对子天听 | 0.0016% |
| 国士无双天听 | ≈ 0% |

> 作为参考：日本麻将"地和"概率约 1/110,000，反推天听概率约 0.01%，与模拟结果在同一数量级。

## 牌编码规则

代码中使用字符串表示牌面：
- **数牌**：`1m` ~ `9m`（万）、`1p` ~ `9p`（筒）、`1s` ~ `9s`（索）
- **字牌**：`1z`(东)、`2z`(南)、`3z`(西)、`4z`(北)、`5z`(中)、`6z`(发)、`7z`(白)

`tenpai.py` 中使用整数编码 `0~33`：
- `0~8`: 1m~9m
- `9~17`: 1p~9p
- `18~26`: 1s~9s
- `27~33`: 1z~7z
